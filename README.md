                          ,,8-bit PingPong''

 To gra komputerowa 2D typu Pong, utrzymana w stylistyce retro (8-bit).
Gracz steruje paletką i rozgrywa mecz przeciwko przeciwnikowi sterowanemu przez komputer (CPU).

Projekt został wykonany w silniku Godot Engine jako aplikacja zaliczeniowa z przedmiotu Aplikacje komputerowe.

Funkcjonalności

rozgrywka jednoosobowa (Player vs CPU),

wybór poziomu trudności przeciwnika CPU, w tym tryb:

Turbo Kozak (najwyższy poziom trudności),

przeciwnik CPU z regulowanym zachowaniem (prędkość i czas reakcji),

system punktacji,

menu główne z muzyką 8-bit,

efekty dźwiękowe (odbicia piłki, punktacja),

samodzielnie wykonane grafiki (pixel art),

modularna struktura kodu.

Wymagania
Wersja desktop:

system Windows,

brak dodatkowych zależności.

Wersja web:

przeglądarka internetowa (Chrome, Firefox, Edge).

Uruchomienie – wersja EXE

Rozpakuj archiwum ZIP z grą.

Uruchom plik:

8-bit PingPong

Gra uruchomi się automatycznie.

Uruchomienie – wersja web

Gra dostępna jest również w wersji przeglądarkowej na platformie itch.io:

 https://itch.io/profile/szpont2004

Wybierz grę i uruchom ją w oknie przeglądarki.

Sterowanie

ruch paletki gracza: W / S lub ↑ / ↓

start / restart gry: przyciski dostępne w menu gry

wybór poziomu trudności: opcja dostępna w menu głównym (tryb Turbo Kozak)

Technologia

Silnik: Godot Engine

Język: GDScript

Grafika: własne sprites (pixel art, 8-bit)

Dźwięk: własne efekty dźwiękowe

8-bit PingPong is a 2D computer game inspired by the classic Pong, designed in a retro 8-bit style.
The player controls a paddle and plays a match against a computer-controlled opponent (CPU).

The project was developed using the Godot Engine as a coursework assignment for the subject Computer Applications.

Features

single-player gameplay (Player vs CPU),

CPU difficulty selection, including the:

Turbo Kozak mode (highest difficulty level),

CPU opponent with adjustable behavior (speed and reaction time),

scoring system,

main menu with 8-bit background music,

sound effects (ball hits, scoring),

self-created graphics (pixel art),

modular code structure.

Requirements
Desktop version:

Windows operating system,

no additional dependencies required.

Web version:

modern web browser (Chrome, Firefox, Edge).

How to Run – Desktop Version (EXE)

Extract the ZIP archive containing the game.

Run the executable file:

8-bit PingPong


The game will start automatically.

How to Run – Web Version

The game is also available as a browser-based version on the itch.io platform:

 https://itch.io/profile/szpont2004

Open the link in a web browser.

Select the game and run it directly in the browser.

Controls

player paddle movement: W / S or Up / Down Arrow keys

start / restart game: buttons available in the game menu

difficulty selection: option available in the main menu (Turbo Kozak mode)

Technology

Engine: Godot Engine

Language: GDScript

Graphics: self-created sprites (pixel art, 8-bit)

Sound effects: self-created sound effects

Music: original 8-bit menu music
